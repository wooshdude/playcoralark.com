Update 1.1.0

Thursday, August 5, 2021

This update is quite juicy. A lot of work went into making the server as smooth as possible. This week was all about a huge migration.

A New Office

  • The computer that hosts the server has changed from running a Windows 10 operating system to a Linux-based one, specifically Ubuntu.
  • Changed the clunky old HDD to a fancy new SSD, which should hopefully keep speed fast and be more consistant than disks.
This is a pretty big deal, considering that the server has never seen anything besides Windows 10. This should also allow for more streamlined updates and hotfixes.

New Things Come And Go

    A new plugin was added that should hopefully attempt to alleviate some lag coming from the multi-gigabyte sized world file that is the server. Because of this, Piglin farms may no longer drop XP in a gracefull stream. Rather, XP will be delivered in meaty chunks.

    Along with this new plugin, we have also opted to remove the "Armored Elytra" datapack, which allowed players to combine their chestplate's resistance and their elytra's ability to soar across the skys. Unfortunatly, with the growing number of players, we deemed the data-pack "Not vanilla enough", so it was scrapped. Elytras that have previously undergone the combination will still remain the same, but you can no longer create more.

    However, elytras did receive a little bit of a buff elsewhere. Slaying the dragon now has a 35% chance to drop a shiny pair of wings. This should hopefully incentivise dragon slaying and should make finding wings much less of a hassle for newer players.

Thanks for continuing to support the server. I'll do my best to keep things updated for you.
