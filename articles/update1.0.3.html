Dynmap Custom Markers

We recently added support for custom markers! As I’m sure you’ve read from our latest TWAC, we want to start incorporating flags into Coralark. This guide will walk you through how to add your own custom flag into the live map.


    Preparing The Image

    Start by designing your flag (A video on good flag design can be found in the latest TWAC). Then export your design as a 32x32 pixel PNG. Make your way to the #minecraft-general section of the discord and upload your image with the command “.upload flag”.

    Note: Please do not abuse this system. This service is specifically for uploading custom markers to the server. Abuse of this system will result in it’s replacement with a manual submission system, which will take much longer.

    Once uploaded, Drew will give you the id for the image.


    Creating The Marker

    Log into the Minecraft server and type the command: 

    /dmarker addicon id:<what you want to name the icon> file:<the code that Drew sent + .png>

    Next, craft the sign and find the place you want to place your marker. If you want to hide it underground, that’s okay. Remember that wherever the sign is placed, that’s where the icon will be on the 3D map.
