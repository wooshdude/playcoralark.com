THIS WEEK AT YOURMOM - 8/26/21


A BREIF HIATUS

-Pringle


This week on the Your Mom SMP, we’re taking some time off.

As most of you know, Minecraft is not the only game we here at yourmom.online like to indulge in. Many of us have several thousand hours invested in “Destiny 2” an FPSMMORPG developed by Bungie. Recently, a new expansion has been released for Destiny, and much of the server is taking a short break in order to return to their gaming roots, and forage for that brand new loot. Because of this, I wouldn’t expect a new TWAYM next week, but you may see some server hotfixes appear.

In the meantime, let’s get into what happened this week!

 

Nothing

In all honesty, we should have done our hiatus this week, since the new expansion came out tuesday. I won’t leave you hanging, though. We’ve got some pretty big plans for the server.

 

Upgrades for All!

The View From Here

  • Since increasing the render distance there hasn’t been any drop in server TPS, which means we’re going to increase the distance from 10 to 12! Photography is going to be a lot easier now, that's for sure!

Cartin’

  • We all know about the JellyCo railway, right? Well, we feel like it deserves a little more love, so in order to compete with the BulletTrain and Elytra, minecarts will be given a significant speed bonus. The details about this will be revealed later, but we’re going to try and implement a system that will still allow for super smelters to function normally.

 

That's all for this week. Like I said, there’s not going to be much in the way of news for a little bit. Hopefully this is enough to satisfy the masses. Until next time, I’ll see you in the Dreaming City.

Glowing Regards

-Pringle
