Yes, that means that once again there’s not a ton we can share about the progress the server has made this week, which is why I ask that if you aren’t currently whitelisted in the server and you’re somehow reading these every week; please get in touch with a server admin and get whitelisted. In the meantime let’s check out what happened in Coralark this week.

 

It’s Raining Men (Hallelujah!)

Animal Farm

  • A large enderman farm has been built in the void surrounding the main end island. Manor co. hopes to use the pearls gathered from this farm to stock the Shop Shop store in the section 8 mall.

Creative World Projects

  • The Emerald Town has expanded their plans for the event hall! Hopefully, it enters the survival world soon.

 

Nothing But Speed!

Now is the part where we get into the technical changes that happened on the server in the background.

Minecarts

  • Minecarts were given a 150% speed boost, which is quite a bit. The downside to this update is that the carts go too fast for the server to catch up. A temporary fix for this is to add powered rails for any hill the minecart is forced to climb, otherwise it gets nervous and bails.

Bottling your Essence

  • You can now store your experience points in Bottles-O-Experience by standing on an anvil. You can then save them for later, store them in your e-chest, or sell them to your friends.
