HOTFIX - 1.0.1


Thursday, July 28, 2021

This update saw the move from the old Spigot platform to the new cool PaperMC platform. This should increase server speeds and prevent it from shitting itself.

Changes

  • Migrated from Spigot to PaperMC.
  • Re-introduced imageOnMap, allowing players to generate custom map art.
    • Splatter maps remain nonfunctional. To be fixed in a later update.

Tweaks

  • Optimized the PaperMC server launcher.
  • Optimized the server's overworld file in an attempt to reduce lag.

Potential Bugs

  • There might be issues with player inventories, but a backup of all inventories has been saved so don't worry.

Website

  • The official website (this one!) received a makeover, turning it into an information hub for the server.
